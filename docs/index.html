Welcome to Ancient Greek NMT

This library provides tools and models for translating Ancient Greek texts using neural machine translation. Whether you're a researcher, student, or enthusiast of classical texts, this library offers powerful tools for working with Ancient Greek translations.

Key Features

  • Pre-trained models for Ancient Greek to English translation
  • Support for multiple translation models including OPUS-MT
  • Easy-to-use Python API
  • Training utilities for custom models
  • Comprehensive evaluation metrics
